Kreativ Mode freischalten

Der Kreativ Mode ist versteckt. Du brauchst den geheimen Code. Der Code liegt auf dem Mond als lila ?-Block. Du kannst ihn jetzt mit Linksklick oder Rechtsklick öffnen.

Status:
Kreativ Mode ist gesperrt.

Wenn der richtige Code eingegeben wird, wird Kreativ Mode aktiviert. Dann ist im Crafting-Buch alles grün, du bekommst keinen Schaden, kannst fliegen und Blöcke werden nicht verbraucht.
